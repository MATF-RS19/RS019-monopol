# :tophat: RS019-monopol
RS019-monopol is an application based on a simple, yet popular board game called "Monopoly".

## Prerequisites
* Qt5
* QtCreator (optional)

## Getting started
Download the repository, then in terminal, run:
```
	$ qmake
	$ make
	$ ./RS019-monopol
```
